# TodoNg

A simple Angular 8 TodoList. 

Data is retrived from a [SpringBoot API](https://github.com/polmic/todo-sb).

Features:
- Display the todos
- Add a todo
- Mark a todo as done
- Create a new todo
- Display a todo's details


## Development server
Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
